# RoR Taichung 官網
RoR Taichung 官網是Rails Taichung為了Rails在台中持續發展深耕並服務社群所建立。本專案也是Rails Taichung第一個團隊開發專案。
* 開發工具為Ruby on Rails
* 專案管理方式以敏捷開發為主，以Trello做為專案管理工具。
